本站的在win下的搭建过程

本站的搭建过程WIn10

安装Node.js、Git、Notepad++
打开cmd
node -v #查看node版本
npm -v #查看npm版本
npm install -g cnpm –registry=http://registry.npm.taobao.org #安装淘宝的cnpm 管理器
cnpm -v #查看cnpm版本
cnpm install -g hexo-cli #安装hexo框架
hexo -v #查看hexo版本
md blog win系统是md新建blog
cd blog 进入blog
hexo init #初始化博客，第一次失败的话重新输入该命令
hexo s #启动hexo服务
启动后重新打开一个cmd窗口，或者Git cmd窗口

修改默认的主题

git clone https://github.com/blinkfox/hexo-theme-matery #下载mstery主题到用户文件夹下面
找到hexo-theme-matery文件夹剪贴粘贴到blog-themes文件夹下，修改文件夹名称为matery

用记事本打开blog文件夹下的_config.yml文件，下拉到最下面，找到theme: landscape,把landscape修改为matery,保存退出
关掉所有cmd 窗口 重新打开一个cmd窗口

cd blog #还是先进blog文件夹
cd clean #清除默认配置数据
cd g 重新部署
cd s 打开服务
这里打开http://localhost:4000/查看博客效果了
